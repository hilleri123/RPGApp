\section{Хедер}
\label{url:root}

\component{Header / Top bar}{cmp:root:header}

\paragraph{Назначение}
Обеспечивает глобальную навигацию по разделам приложения и доступ к профилю пользователя.

\screenshot{components/header.png}{Хедер с навигацией}

\action{Переход по логотипу}{act:root:header:logo}
\uilink{Логотип ``RPG Master''}{/}{ui:root:logo}
\paragraph{Действие}
Открывает главную страницу приложения (\texttt{/}), см. \hyperref[url:root]{главу ``Главная''}. % \ref{url:root}

\action{Переход: Главная}{act:root:header:nav-home}
\uilink{Пункт меню ``Главная''}{/}{ui:root:nav-home}
\paragraph{Действие}
Открывает главную страницу (\texttt{/}), см. \hyperref[url:root]{главу ``Главная''}. % \ref{url:root}

\action{Переход: Правила}{act:root:header:nav-rules}
\uilink{Пункт меню ``Правила''}{/rules}{ui:root:nav-rules}
\paragraph{Действие}
Переходит в раздел правил (\texttt{/rules}), см. \hyperref[url:rules]{главу ``Правила''}. % \ref{url:rules}

\action{Переход: Сценарии}{act:root:header:nav-scenarios}
\uilink{Пункт меню ``Сценарии''}{/scenarios}{ui:root:nav-scenarios}
\paragraph{Действие}
Переходит в раздел управления сценариями (\texttt{/scenarios}), см. \hyperref[url:scenarios]{главу ``Сценарии''}. % \ref{url:scenarios}

\action{Переход: Группы}{act:root:header:nav-access-groups}
\uilink{Пункт меню ``Группы''}{/access_groups}{ui:root:nav-access-groups}
\paragraph{Действие}
Переходит в раздел управления доступом: группы и пользователи (\texttt{/access\_groups}), см. \hyperref[url:access-groups]{главу ``Группы''}. % \ref{url:access-groups}

\action{Переход: Профиль}{act:root:header:nav-me}
\uilink{Ссылка на профиль пользователя}{/me}{ui:root:nav-me}
\paragraph{Действие}
Открывает страницу профиля пользователя (\texttt{/me}), см. \hyperref[url:me]{главу ``Профиль''}. % \ref{url:me}

\action{Открыть мобильное меню}{act:root:header:mobile-menu}
\uibutton{Кнопка меню (``бургер'')}{ui:root:btn-mobile-menu}
\paragraph{Действие}
Открывает/закрывает мобильное меню навигации (на малых экранах).
